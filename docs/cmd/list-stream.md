# nestos-installer list-stream

## Description

List available images in a NestOS stream

## Usage

**nestos-installer list-stream**

## Options

| **--stream**, **-s** *name* | NestOS stream [default: stable] |
| **--stream-base-url** *URL* | Base URL for NestOS stream metadata |
